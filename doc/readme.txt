Git-Flow 有以下几种分支：
feature 分支：开发者进行功能开发的分支。
develop 分支：对开发的功能进行集成的分支。
release 分支：负责版本发布的分支。
hotfix 分支：对线上缺陷进行修改工作的分支。
master：保存最新已发布版本基线的分支。
